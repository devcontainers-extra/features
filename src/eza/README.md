# eza (via Github Releases) (eza)

eza is a modern replacement for ls.

## Example Usage

```json
"features": {
    "ghcr.io/devcontainers-extra/features/eza:1": {}
}
```

## Options

| Options Id | Description | Type | Default Value |
|-----|-----|-----|-----|
| version | Select the version to install. | string | latest |
